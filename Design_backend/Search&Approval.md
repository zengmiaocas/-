### 开发目标

实现项目搜索筛选、申请与审批管理功能，完成组队业务闭环。

### 核心任务

1.  项目搜索筛选：支持按项目名称 /
    描述、标签进行关键词搜索，动态刷新大厅数据。

2.  申请管理：用户申请加入项目、撤销申请，队长查看待审批申请。

3.  审批管理：队长同意 /
    拒绝申请，同意后自动发送群欢迎消息，满员自动截止招募。

4.  申请记录管理：支持用户 / 队长隐藏已处理的申请记录。

### 核心接口实现

  ---------------------------------------------------------------------------------------
  接口路径              功能           关键逻辑
  --------------------- -------------- --------------------------------------------------
  /api/dashboard_html   动态数据接口   接收搜索关键词，返回筛选后的项目、申请、审批列表
                                       HTML 片段

  /apply                申请加入接口   防重复申请，支持被拒绝 / 移出后重新申请

  /cancel_apply         撤销申请接口   将申请状态改为 "已取消"，隐藏记录

  /audit                审批接口       处理队长同意 /
                                       拒绝操作，更新申请状态，发送系统消息

  /hide_record          隐藏记录接口   标记申请记录为不可见，仅用户 / 队长端隐藏

  /api/mark_read        标记已读接口   标记队长审批、用户申请结果为已读，清除红点提醒
  ---------------------------------------------------------------------------------------
